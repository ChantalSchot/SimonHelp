How-to-use

Buttons

Clear: Clears the current input from the search bar.

Search again: Refreshes the search with the current search term(s).

Show zero contributions: Uncheck to show only ingredients that are currently selected (thus have a contribution of at least 1%). Check to show complete list of ingredients.

Increase contribution by one (Enter): Increase the contribution of the currently selected ingredient by 1%.

Set contribution to zero (Ctrl+Z): Sets the contribution of the currently selected ingredient to 0% (removing it from the feed), regardless of the previous contribution value.

Σ= 100%: Increases contributions of currently selected ingredients to rate so the sum adds up to 100 (thus total contribution of 100%).

OK: Confirms ingredient selection and returns you to the feeds window. Undo changes through X in the top right corner, or through "Restore" in the feeds window.

Fields

Protein sources and their data are presented in 5 columns:

Name protein source
get a drop-down list by clicking in this field and select a defined protein source.

Contribution to feed
enter the percentage contribution of the protein source to total feed protein (not to the feed).

Endogenous protein factor
default multiplication factor for the protein source selected, if it would be the only protein source used.
This multiplication factor is set to one for milk proteins, and generally is somewhat higher for vegetable protein sources.
You cannot change this factor, but after leaving this window, a weighted mean is calculated, which you can adapt in the feed window (amino acid sheet).

Protein factor
gives the crude protein factor (g crude protein/g nitrogen) for the selected protein source. This value is fixed.

True digestibility coefficient protein
gives the true digestibility coefficient for the selected protein source.
You cannot change this value.
After leaving this window, a weighted mean is calculated for the protein sources selected, and used as a default.
You can adapt this value if you like.

Functionality of the protein sources window

In this window, you can specify your protein sources.
You can choose between skimmed milk powder, whey, soy protein isolate, soy protein concentrate, soluble wheat gluten protein and potato protein.
The sum of the contributions has to add up to 100%.
Based on these data, Simon generates default values for protein, fat, starch and lactose digestibility, the protein to nitrogen ratio, the amino acid composition and the endogenous protein factor.
You can change these default values when you return to the feed window.
The endogenous protein factor affects energy metabolism and the apparent faecal protein digestibility, which can be retrieved as an output variable (!) in the output window.
