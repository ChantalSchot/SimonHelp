How-to-use

Buttons

Clear: Clears the current input from the search bar.

Search again: Refreshes the search with the current search term(s).

Show zero contributions: Uncheck to show only ingredients that are currently selected (thus have a contribution of at least 1%). Check to show complete list of ingredients.

Increase contribution by one (Enter): Increase the contribution of the currently selected ingredient by 1%.

Set contribution to zero (Ctrl+Z): Sets the contribution of the currently selected ingredient to 0% (removing it from the feed), regardless of the previous contribution value.

Σ = 100%: Increases contributions of all selected ingredients to ratio so the sum adds up to 100 (thus total contribution of 100%).

OK: Confirms ingredient selection and returns you to the feeds window. Undo changes through X in the top right corner, or through "Restore" in the feeds window.

Fields

Ingredients are listed in a table consisting of three columns:
Identifier: Name of the ingredient.

Description: Description of the ingredient.

Contribution: Percentage contribution of the ingredient to total feed composition.

How-to: Adding ingredients to a feed

In the ingredients window, you can select various ingredients that form the basis of a feed. This works the same for milk replacers, concentrates and forages, although different ingredients may be listed for the different feed types.
  • To add an ingredient, find it in the alphabetical list either by scrolling through or using the search function.
  • Increase the contribution in the right column to a value between 1-100 (this is the percentage of the ingredient in the feed).
  • Select all ingredients and uncheck Show zero contributions (top right) to show only selected ingredients in the table.
  • Ensure that the sum of all contributions adds up to 100, or let SiMON calculate this by clicking on Σ = 100%.
  • Once the list is complete and correct, click ✔ Ok to save the ingredient composition.
Based on the selected ingredients, SiMON will generate default values for nutrients, degradation rates and amino acid composition. These values can be adjusted afterwards if desired. Note that changing the ingredients will regenerate the default values, discarding manual changes made previously.
