How-to-use

Open the Output Ranges window through Menu bar → Settings → Output Ranges. Predefined output ranges can be used to view results of a simulation in the Output window per range (weight or time).

Buttons

New: Define a new range definition (max 16 characters).

Copy: Copy the selected range definition.

Delete: Delete the selected range definition.

Save: Save a newly defined or changed range definition.

Restore: Undo changes since last save.

Close: Close the Range Definition window.

Fields

Name: Enter a name for the range definition.

Description: Enter a description for the range definition.

Based on: Define whether range definition is based on time (days) or weight (kg).

Endpoints: Table to add different range steps.

How-to: Creating or adjusting output ranges

In the Output Ranges window, you can create range definitions for output ranges of the results. There are two predefined range definitions, namely:
  • TimeRangeWks: Fixed time step of 7 days for 10 weeks.
  • WeightRange10kg: Fixed weight step of 10 kg.
To create a new range definition, either click on the new button or select an existing range definition and copy it. Enter a name and description, choose whether it is based on time or weight, and fill in the endpoints (rounded numbers). To add a new endpoint, place the cursor on the last row and press ↓ (down arrow; only works if the last row has been filled out), the list will automatically be sorted in ascending order. SiMON assumes that the first defined endpoint is always day 0 or 0 kg. The last endpoint does not necessarily match the end of a simulation. Once all endpoints have been filled out, click ✔ Save and the range definition can be used in the Output window. Adjust an existing range definition by selecting and changing the endpoints are already filled in. Delete an endpoint with <Ctrl + Del>.

Important note
Keep in mind that averages are calculated on a time basis. If you specify very small ranges (e.g. only a few kgs), the averages over the weight ranges will be calculated based on the days closest to the desired start and end points. A small defined range may thus be smaller than how much the calf grows in one day.
