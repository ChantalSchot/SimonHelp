How-to-use

Buttons

Copy: Define a new genotype by copying one of the three basic genotypes.

Delete: Delete a genotype (not possible for the three basic genotypes). Simulations that make use of deleted genotype won't work any more. You can change an existing genotype and save it, but previous simulations won't produce the same results after this.

Save: Save a newly defined or changed genotype.

Restore: Undo changes since last save.

Fields

Name: Choose a name for your genotype (max 15 characters)

Description: Choose an appropriate description

Based on: When defining a new genotype, you will see here which of the three basic genotypes your new one is based on.
The three basic genotypes differ in many parameters, not all of which you can see and change.

Maximum rate of amino acid oxidation: Change the default multiplier within the limits.
Increasing this factor increases the priority of amino acids for oxidation and will therefore decrease protein deposition and increase fat deposition rates.

Maximum rate of fat synthesis: Change the default multiplier within the limits.
Increasing this factor increases the priority of fatty acids for fat synthesis and therefore increases fat deposition rate and decreases the utilization of fatty acids for oxidation.

Maximum rate of fatty acid oxidation: Change the default multiplier within the limits.
Increasing this factor increases the priority of fatty acids to be used for oxidation (energy supply) and will lead to a decreased fat deposition and possibly less oxidation of other nutrients.

Maximum rate of fatty acid synthesis: Change the default multiplier within the limits.
Increasing this factor increases the priority of Acetyl-Coenzyme A (Acetyl-CoA) to be used for fatty acid synthesis. It will indirectly increase fat deposition and possibly decrease protein deposition.

Energy requirement for maintenance processes: Change the default multiplier within the limits.
Increasing this factor increases energy expenditure on maintenance processes and will therefore reduce protein and fat deposition.
Please note that differences in maintenance energy expenditure between genotypes, as far as related to changes in body composition, are already accounted for by SiMON. Changing this multiplier is useful to account for differences in environment (e.g. ambient temperature, group housing systems, etc.).

How-to: Creating new genotypes

In the genotype window you can create new genotypes based on the three (basic) genotypes for which SiMON is parameterized:
  • Dutch Black-and-White male calves: 70% Holstein-Friesian x 30% Dutch Friesian
  • Dutch Red-and-White male calves
  • Dutch Black-and-White male calves (improved)
To create a new genotype, you have to select an existing genotype and copy it, which duplicates initial body composition and a complete set of parameters from the existing genotype.

You can change five model parameters and consequently some of the nutrient partitioning rules of your genotype. These model parameters are multiplication factors and are set to 1 for the Dutch Black-and-White male calves, but have varying default values for the Red-and-White calves. Keep in mind: Setting default values from Red-and-White calves to 1 will not give you a Black-and-White male genotype, due to other parameter differences you cannot change. It should be noted that SiMON was originally calibrated for Dutch Black-and-White male calves, and the results obtained with this genotype are more reliable than with the other genotypes.
