SiMON - Print / Report

SiMON content list


  • In every window you can create a report by pressing Print (Menu bar → File → Print...).
  • In most windows you will get a print of the selected item.
  • In the output windows, a report is generated depending the sub-window you're working in.
  • Reports are in HTML format can be viewed, printed (Ctrl+P), saved (Ctrl+S) or copied to clipboard (Ctrl+C).
