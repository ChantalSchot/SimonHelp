Once the simulation has run, the results can be viewed in the Output window. At the top of this menu, there are three options to view the output: at the end of the simulation, per range (time or weight) and on times within day(s).

Extra options:
Graphs: How to make graphs from the output table.
Output Ranges: Define output ranges for the output window (in days or weight).
Variable Groups: Define variable groups that can be selected in the output window (per range or times within day(s) only).
List of Variables: A complete list of available output variables and descriptions.

Important note
Depending on your choice of viewing output per time/weight range you may or may not see all variables. For example, live weight gain (LWG, g/d) can be calculated every time step, but not as an average over a certain time or weight range. In that case, you will have to select the variable average live weight gain (avgLWG, g/d). All variables starting with avg (average) or cavg (cumulative average) are only available as range-output. A variable group can consist of several types of variables, but selecting this variable group will only show the results of the variables that correspond with your choice of output range or step.

Output at the end of the simulation

Results shown at the end of simulation are a list of selected variables as calculated at the end of the simulation, most representing slaughter data. Output of up to four specified simulations is presented in four columns, one for each simulation. The results can be exported using Menu bar → File → Print... (in html format). This list of output variables cannot be adjusted by the user; more tailor-made information and specific variable groups can be viewed in the output per range or times within day(s).

Output per range (time or weight)

Results shown at the output per range (time or weight) are shown for different Variable Groups. Every variable represents a new column and every record (row) represents a step in the defined time or weight range. If multiple simulations are shown in the same output table, each variable column shows results for each simulation respectively. For example for empty body weight, the first column of Q ew (1) corresponds to simulation 1, the second column of Q ew (2) shows simulation 2, etc.

Preferences for presenting results
Above the output table, there are various options to adjust how the results are shown. Changing these options is possible without re-running the simulations. After changing one or more of the options, make sure to click Update so the changes are applied to the table.
Simulation: Check or uncheck simulations that are shown in the table.
Output ranges: Select a fixed time step (in x days) or averages for a time or weight range (see Output Ranges).
Variables: Show a predefined group of variables (see Variable Groups).
Graph: Create a graph of the results (see Graphs).

Output as times within day(s)

Results shown as times within day(s) are shown for different moments throughout the day: the time step between each record is 10 seconds. Variables are shown in a similar way as in the Output per range (see section above). In this window, the time step cannot be defined, but it is possible to define the first and last day, showing all time steps of 10 seconds for each of those days in between. Output shown as times within day is useful to see daily fluctuations in variables. Other preferences for output are similar to the output per range (see above).
