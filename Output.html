Output at the end of a simulation

In this window you can view the simulation outputs of up to 4 simulations (see simulation window).
Only the end-point variables will be shown, which are basically slaughter data.
Results of the different simulations are presented in different columns.
You cannot change anything in this window. For tailor-made output, view the output at different time steps/ranges.
You can export or print this output.

Output per range (time or weight)

In this window you can view the simulation results of up to 4 simulations (see simulation window).
The results are shown in the window below and are sorted by the selected variables.
The results of the different simulation runs (1 to 4) are presented per output variable in separate columns.
The number of the simulation follows the name of the output variable: LW[1] means live weight of the first simulation run, LW[4] means live weight of the fourth simulation run.

Setting your preferences for presenting simulation results

At the top left you can switch the results of a particular simulation run on or off.
You can repeatedly make changes without having to re-run your simulations.
You will, however, have to press the Update button before the output table is actually changed.

Defining the time ranges

You can choose how to view your output variables:

  • per day                                      --> choose fixed time step with value 1
  • every x days                              --> choose a fixed time step with value x
  • average per defined time range --> select an existing time range from the drop-down list
  • graphically                                 --> choose a fixed time step of 1 and make a graph

If you have not defined any output time ranges, you can do so in main menu, settings (Output ranges).

Selecting the presented output variables

Select an existing variable group from the drop-down list and click Update.
If you decide to view another variable group, select it and press Update again.
You don't have to re-run the simulations.

If you have not yet defined variable groups, you can do so in main menu, settings (Variable groups).

Important

Depending on your choice of viewing output per time step or time range you may or may not get to see certain variables.
Live weight gain (LWG, g/d), for example, can be calculated every time step, but not as an average over a certain time or weight range.
In that case, you will have to select the variable average live weight gain (avgLWG, g/d).
All variables starting with avg (average), cavg (cumulative average) are only available as range-output.
A specific variable group can consist of several types of variables, but selecting this variable group will show the results of the variables that correspond with your choice of output range or step.
