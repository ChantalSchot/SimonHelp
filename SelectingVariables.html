How-to-use

Open the Variable Groups window through Menu bar → Settings → Variable Groups. Predefined variables groups can be used to view results of selected variables in the Output window.

Buttons

New: Define a new variable group (max 15 characters).

Copy: Copy the selected variable group.

Delete: Delete the selected variable group.

Save: Save a newly defined or changed variable group.

Restore: Undo changes since last save.

Close: Close the Variable Group window.

↑ and ↓: Move a selected variable up or down in the list (order how results are shown in output table).

> and <: Add or remove a variable into or out of the "Selected variables" list.

>> and <<: Add or remove all listed variables into or out of the "Selected variables" list.

Fields

Name: Enter a name for the variable group.

Description: Enter a description for the feed.

Per time step/range: Show only variables that are visible in the results per day or per custom range.

All variables (drop-down): Shows a list of different variable categories.

Available variables: Shows available variables that are not in the selected variable group.

Selected variables: Shows a list of selected variables in the current variable group.

How-to: Creating or adjusting variable groups

In the Output window, you can choose to show specific variable groups in the table of results. There are several predefined variable groups, namely:
  • Body weight: Includes variables related to body weight composition (including organs).
  • Feeding: Includes variables related to feed intake of different feeds.
  • N balance: Includes variables related to nitrogen balance.
From the complete list of available variables you can create your own selections by using variable groups. To create a new variable group, either click on the new button or select an existing variable group and copy it. Enter a name and description, choose whether it is based on time step or range, and select variables that you want to add to the variable group. Move a variable to the selected variables in the variable group by pressing > or remove a selected variable by pressing <. If you want to move all variables from the list to selected or unselected, use the >> or << buttons. Use the red ↑ and ↓ arrows to change the sequence in which variables are shown in the output window. Click ✔ Save and the variable group can be used in the Output window.
