Graph

In Simon, you can have several graphs open at the same time.
For optimal use of the graph options, it is important to read the following paragraphs carefully.

Making a graph

  • you define a graph in the output window (per time step/range)
  • you can only make graphs when you selected a fixed time step of 1 day (and subsequently pressed Update to update the table)
  • time will always be the X-variable.
  • a graph is always based on the output you saw in the table in your output window at the time making the graph.
    You can only plot variables that occur in that list.
  • you can create more graphs (and have them on your screen simultaneously).
    These graphs can, but don't have to be, based on the same table.
  • graphs you make will be numbered automatically (graph1, graph2, etc.)
  • all graphs can have only one Y-axis
  • each graph will carry its own settings.
    You can change these by clicking top-left on "graph" and then "settings".
  • by default, only the first two variables in the output list will be plotted in the graph. you can change this, as described above.
    You can also adapt the sequence in the variable list and make sure the ones you would like to see are on top.
If you would like more information, see advanced graph settings
