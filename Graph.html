Making a graph
Advanced graph settings

In SiMON, you can have several graphs open simultaneously. For optimal use of the graph options, read the information below.

Creating a graph

  • A graph can only be created for output per range or times within day(s).
  • Make sure the output table is updated before creating a graph.
  • Create a graph by clicking Graph on the top right above the output table (not visible at end of simulation).
  • Time is shown on the X-axis and each graph can only have one unit on the Y-axis.
  • Specify the Y-axis variable(s) by clicking Menu bar → Graph → Settings, then select variables you wish to see.
  • By default, only the first two variables from the output are shown. See advanced settings below for more information.
  • Multiple graphs can be made simultaneously (see advanced settings below).
  • Each graph can be adjusted separately and will have its own settings.

Advanced graph settings

Open graph settings by clicking Menu bar → Graph → Settings.

Graph layout
In the settings window, the basic layout elements of the graph can be adjusted manually by changing:
  • Legend, scale and range of X-axis and Y-axis
  • Location of the legend
  • Description of the legend
  • Visibility of background grid
  • Graph title
Numerous advanced layout options can be adjusted (e.g. fonts, title, ...) by clicking More options... at the bottom right corner of the settings window (these are not discussed here further).

Variable selection
Variables can be selected by checking or unchecking the "visible" column in the Settings list. If a variable is not available, the output needs to be updated (for example, choose another variable group) so you can create a new graph that contains the variable you wish to plot. Line style and color of each variable can be adjusted by clicking on the corresponding line (in the settings window), which allows manual adjustment of style, colour and thickness.

Multiple graphs
Multiple graphs can be made at the same time. Minimize (do not close) a graph, update the output table and create a new graph to have two graphs at the same time. By clicking Menu bar → Windows → ... you can choose to show multiple graph windows as overlapping or adjacent to each other, or just drag them where you want. New graphs are numbered sequentially (e.g. Graph 1, Graph 2, Graph 3, ...) as long as the previous graph has not been closed. Layout and settings can be adjusted for each graph separately.

Export options
Graphs can be exported or saved by clicking Menu bar → Settings → ...
  • "Print..." allows printing or saving the graph as pdf (Print to PDF).
  • "Save as" allows exporting the file as bitmap or meta file.
  • "Copy" will copy the graph, allowing you to paste it in another program.
