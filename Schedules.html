How-to-use

Buttons

Similar for milk replacers, concentrates and forages.
New: Define a new feeding schedule.

Copy: Copy the selected feeding schedule.

Delete: Delete the selected feeding schedule.

Save: Save a newly defined or changed feeding schedule.

Restore: Undo changes since last save.

Fields

Name: Enter a name for the feeding schedule (max 15 characters).

Description: Enter a description for the feeding schedule.

Step size: Enter a step size (in days) on which to add changes in the feeding schedule (gram/day). Using Ctrl+D in the table will add a new row (with specified step size) in the schedule.

Specification of feeding schedule:
Day: Starting day on which the specified quantities will be fed.
Times (milk replacers only): Daily time schedule for when milk replacer is fed during the day (see Feeding Times).
Tot.: Total quantity of feed given per day.
Feed 1 - 6: List of predefined feeds (specified in the Feed window) composing the daily feed intake.
g/d (1 - 6): Quantity of feed given per day (gram/day, specified for each feed separately).

How-to: Creating and adjusting schedules

Similar for milk replacers, concentrates and forages.

In this window you specify the quantity of each feed given on each point in time to the simulated calf. To create a new schedule, either click on the new button or select an existing feed and copy it. Define the name, description and step size for the feeding schedule and place the cursor in the first record of the table (Specification of feeding schedule). Every record (row) in the table defines a feeding schedule for a period of time, starting at the specified 'day' for that record and ending on the day of the next record.

For each record, enter a starting day, select one or more feeds (up to 6, defined in the Feed window) and fill in the corresponding quantities (in g/day) that should be fed each day. For milk replacers only, the daily feeding times (column "Times") should be specified as well, more information about this can be found here: Feeding Times. Press Ctrl+D to add a new record and repeat the steps. For example, if the first record starts on day 0 and the second record starts on day 9, then the feeds from the first record will be applied during days 1 - 8. The feeds from the second record are applied starting on day 9. Once all records are filled out, click ✔ Save and the schedule becomes available for simulations.

If you choose to simulate until a target weight (see Simulation window) then the simulated calf may reach the target weight before the complete feeding schedule is finished. If this is not the case, the last record of the feeding schedule is applied until the end of the simulation.

Tips for creating records in a feeding schedule

  • Use <TAB> to enter the fields in the table from left to right.
  • Delete the selected record (row) with <Ctrl + Del>.
  • Select a feed by pressing <space> and subsequently  <Alt + ↓ > (arrow down), then
    use the arrow keys to select the feed and press <enter> to confirm.
